# etyl-touchjoystick
A Touch Joystick to Etyl

### Maven
```
<dependency>
    <groupId>com.harium.etyl</groupId>
    <artifactId>touchjoystick</artifactId>
    <version>1.0.0</version>
</dependency>
```

### Gradle

```
compile "com.harium.etyl:touchjoystick:1.0.0"
```
